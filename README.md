
##  Aliases

* ```mkymd``` - Create a folder named today's date eg: 2020-02-22
* ```sau``` - Shortcut to upgrade debian based systems
* ```a2path``` - Exports current directory to $PATH
* ```gh``` - Search bash history for command
* ```rebash``` - Reload bashrc
